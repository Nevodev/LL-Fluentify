# LL-Fluentify
为QQNT添加 真 亚克力样式的 右键菜单 / 表情面板。需要安装 **[LiteLoaderQQNT](https://github.com/mo-jinran/LiteLoaderQQNT)**

已适配深色模式与浅色模式。

## 安装
下载 Zip 包并解压 (文件列表右上角点击 Code > Download ZIP)，将解压出的文件夹移动至 `LiteLoaderQQNT数据目录/plugins/` 内，并重启 QQ.
